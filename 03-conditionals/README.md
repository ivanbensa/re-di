# JavaScript Lesson 3 - Practice Tasks

## Task 1

Create a variable age.
If age is 18 or more, show:

```js
"You can enter"
```

## Task 2

Create a variable temperature.
If temperature is greater than 20, show:

```js
"It's warm outside"
```

## Task 3

Create two variables:
- hasTicket
- age

If age is 18 or more AND hasTicket is true, show:

```js
"You can enter the cinema"
```

## Task 4

Create a variable color.
If color is `"red"` OR `"orange"`, show:

```js
"Warm color"
```

## Task 5

Create a variable isDaytime.

If NOT daytime, show:

```js
"It's nighttime"
```

Create variables:
