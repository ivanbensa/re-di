## Task 2 - Shopping Time

Create a shopping system.

Buy in this order:
1. Milk
2. Cheese
3. Bread
4. Butter (only if bread was bought)

After every purchase:
- decrease the budget
- output what was bought

At the end:
- output remaining budget

Example variables:

```js
let budget = 10;

let milkPrice = 2;
let cheesePrice = 4;
let breadPrice = 2;
let butterPrice = 1;
```
